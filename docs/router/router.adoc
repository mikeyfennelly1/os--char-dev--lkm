= router

This module redirects the output of the job to either the /proc filesystem, or a userspace stream.

== The router API

=== Change the standard output to /proc

[source, c]
----
#include "router.h"

int main()
{
    if (router(PROC) < 0)
    {
        printf("Failed to redirect output to /proc file\n");
    }
}
----
